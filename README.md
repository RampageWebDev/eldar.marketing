# eldar.marketing

Варианты логотипа для маркетингового агентства.

## Варианты

### 1. Signal
Стилизованная буква E с акцентной точкой. Символизирует импульс и результат.

### 2. Target  
Концентрические круги с центральной точкой. Символ точности и фокуса на цели.

### 3. Nexus
Сеть связанных узлов. Отражает системный подход и взаимосвязи в маркетинге.

## Просмотр

Откройте `assets/eldar-marketing/preview.html` для просмотра всех вариантов.

## Технические характеристики

- **Шрифт:** Inter SemiBold (Google Fonts)
- **Основной цвет:** #1A1A2E
- **Акцент:** #0066FF
- **Форматы:** SVG, PNG
- **Минимальный размер:** 120px по ширине

## Использование

Каждый логотип представлен в двух вариантах:
- Для светлого фона
- Для темного фона

Все файлы в векторном формате SVG для масштабирования без потери качества.
